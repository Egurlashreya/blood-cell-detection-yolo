# Blood Cell Detection using YOLO

A computer vision project that detects and classifies Red Blood Cells (RBC), White Blood Cells (WBC), and Platelets from microscopic blood sample images using the YOLO object detection model.

## Features

* Blood cell detection from medical images
* Classification of RBC, WBC, and Platelets
* Image preprocessing and annotation handling
* Real-time inference support
* Bounding box visualization

## Tech Stack

* Python
* YOLO
* OpenCV
* NumPy
* Pandas
* Matplotlib

## Workflow

1. Collect and preprocess blood cell image dataset
2. Annotate and organize training data
3. Train YOLO detection model
4. Perform inference on test images
5. Evaluate detection accuracy

## Applications

* Medical image analysis
* Automated blood diagnostics
* AI-assisted healthcare systems
